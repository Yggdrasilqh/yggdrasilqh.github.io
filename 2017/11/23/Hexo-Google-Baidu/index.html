让百度和Google收录您的网站

发表于 2017-11-23 | 分类于 Computer Technology

摘要

在上一片文章中我们搭建好了自己的博客，但如果连搜索引擎都找不到，这逼还怎么装？

百度收录

在开始之前应该先看看他们是否收录了我们的网站，百度site:yourBlog.domain.com。

如果出现这个，那么就是它还没有收录。不收就算了，自己提交。

点击下方的提交网址链接

在这里我发现他的建议似乎是正确的，先验证再提交吧。

点击验证网站所有权，填写你要验证的网址。最简单的方法就是文件验证，通过下面的说明可以下载验证文件。根据它的提示我们把文件放到域名（即GitHub或Coding的Pages项目）的根目录下。怎么放大家都应该知道，然后点击确认。

显示验证成功后就可以在站点管理里看到刚才添加的站点了。

但是现在百度还不知道我们网站里有一些什么网页，所以需要生成一个sitemap给它。

什么是sitemap呢

Sitemap 可方便网站管理员通知搜索引擎他们网站上有哪些可供抓取的网页。最简单的 Sitemap 形式，就是XML 文件，在其中列出网站中的网址以及关于每个网址的其他元数据（上次更新的时间、更改的频率以及相对于网站上其他网址的重要程度为何等），以便搜索引擎可以更加智能地抓取网站。

听起来挺简单，做起来更简单。已经有人做过这个插件啦，现在只需要在Blog根路径下

安装成功以后我们在每次的hexo g后在public文件夹下就会自动生成我们需要的baidusitemap.xml文件了。

这时只需要在刚才的页面（百度站长）右侧的sidebar里选择链接提交，下方自动提交的输入框里输入yourblog.domian.com/baidusitemap.xml就行啦，随后百度就会不定时的抓取你的网站链接。可他在提示里有一句

百度站长平台为站长提供链接提交通道，您可以提交想被百度收录的链接，百度搜索引擎会按照标准处理，但不保证一定能够收录您提交的链接。

所以就是我提交了它如果觉得太辣鸡还不收？？

令人开心的是我提交之后已经过了三天它还是没收。

可人家Google都把我收了！

不收拉到（听有人说百度收录速度比较慢，大概需要一个月的时间）
